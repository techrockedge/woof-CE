The new release of Dpup Bullseye is out!

Features include:

  • Zstd compression for Squashfs images, for faster decompression
  • Firefox, Sylpheed and Transmission running as spot
  • The firewall is enabled by default
  • Support for compressed RAM swap devices
  • Much improved font rendering
  • A renewed and improved Dark Gray theme
  • Ad-blocking, enabled by default
  • DNS cache for faster browsing
  • GTK+ 3 upgrade of almost all applications and improved tray icons
  • Faster boot
  • Firefox, with privacy-oriented default settings and lower resource consumption
  • GTK+ 3 theme support
  • A subset of the Liberation font set, for better-looking websites
  • PulseAudio and pa-applet
  • The retro 431 theme, reminiscent of Puppy Linux 4.3.1
  • A new terminal on Ctrl+Alt+T
  • Support for bash command-line completion
  • Support for Wi-Fi rescanning
  • Cleaner and translation-ready menus
  • Consistent theming for applications running as spot
  • Increased backwards-compatibility with old GTK+ 2 gtkdialog applications
  • Keyboard shorcuts to maximize, minimize and snap windows
  • PulseAudio support for applications running as spot
  • The working directory in the shell prompt
  • Restored adrv support and support for automatic adrv redirection
  • gexec on Alt+F2
  • vi as the default editor
  • Tear-free rendering and faster startup using Cage and Xwayland
  • And much more!
