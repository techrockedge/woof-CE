Keyboard Shortcuts

  • Alt+Tab switches to the next window
  • Alt+Shift+Tab switches to the previous window
  • Alt+F1 opens the menu
  • Alt+F2 runs a command
  • Alt+F4 closes a window
  • Super+Up maximizes a window
  • Super+Down minimizes a window
  • Super+Left snaps a window to the left
  • Super+Right snaps a window to the right
  • Alt+= maximizes a window
  • Alt+- minimizes a window
  • Alt+] snaps a window to the right
  • Alt+[ snaps a window to the left
  • Super+d shows the desktop
  • Super+l locks the screen
  • Super+r runs a command
  • Ctrl+Alt+t starts a new terminal
  • Ctrl+Alt+Delete starts the task manager
  • Ctrl+Alt+Right switches to the next desktop
  • Ctrl+Alt+Left switches to the previous desktop

Common Issues

Package Management

To use the apt package manager, one must first download a list of available packages:

apt update

For example, to find information about GIMP:

apt search gimp

To install it:

apt install gimp Support for this package manager is experimental; some packages will not work, some applications refuse to run as root, and some packages may break the Vanilla Dpup installation. It is recommended to create a backup before installing a package, especially if it has many dependencies. If left unused, apt poses no risk to a Vanilla Dpup installation.

Graphics and Input

On supported hardware, the default X server is Xwayland and not the traditional X.Org. This may cause issues, especially when using multiple monitors or running 3D games. In addition, desktop environments like Xfce and KDE work, but window manager level compositing is disabled, because the Wayland compositor already handles that. Therefore, functionality of applications that rely on X-level compositing (for example, terminals with a transparent background) is degraded.

To force use of X.Org, go to Setup -> Graphics / Screen and click Change to X.Org.

Blurry Text

Hinting is disabled by default. This makes text faster to render, but small text may be light or blurry.

To use the "slight" hinting style, which is often the default in other distributions or desktop environments, run the following commands in the terminal:

rm -f /etc/fonts/conf.d/10-hinting-none.conf ln -s /usr/share/fontconfig/conf.avail/10-hinting-slight.conf /etc/fonts/conf.d/ ln -s /usr/share/fontconfig/conf.avail/10-autohint.conf /etc/fonts/conf.d/
